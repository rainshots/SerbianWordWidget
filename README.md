# SerbianWordWidget
Widget for learning new words.
Для тех кто изучает Сербский и много сидит за пк (windows), я сделал виджет, который позволяет фоново изучать слова. 
Это небольшое окошко поверх всех окон, которое показывает слово и его перевод на русский и английский (его тоже можно заодно подтянуть).

Слова автоматически меняются раз в несколько часов (настраивается). 
Логика такая — целый день видишь на экране слово и постепенно его запоминаешь. Например, 1 день - 1 слово. За год это уже более 300 слов!.

В списке слов изначально 300 слов. Однако можно редактировать список слов как пожелаете (правой кнопкой по окну показывает все возможные опции и настройки).
Просмотренные слова помечаются в файле "1". Не просмотренные "0". Виджет отображает первое слово с "0" с начала списка со словами файла Words (файл лежит в C:\Users\USERNAME\AppData\Roaming\SerbianWordWidget).
Виджет должен запускаться автоматически при запуске Windows.
